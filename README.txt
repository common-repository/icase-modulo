== Description ==

Este módulo faz a integração entre o seu portal e o sistema iCase, promovendo o acesso restrito a usuários associados à sua entidade. Ele também oferece a Área do Associado, onde o usuário pode visualizar o seu perfil e outras informações.

= Como usar =

* No item 'iCase' localizado no menu lateral na área administrativa do WordPress, você irá inserir no campo 'Sigla do cliente' a sigla da sua entidade, fornecida pela Itarget Tecnologia;
* Posicione no seu tema o widget 'iCase login', para que o formulário de login seja exibido.
* Será criada uma página de apoio chamada 'Espaço do associado', com links para informações do associado;
* Utilize também o botão 'iCase - Conteúdo restrito' no editor de texto, em suas páginas e posts, para inserir o shortcode do conteúdo restrito aos associados;

== Installation ==

1. Faça upload do plugin para o diretório '/wp-content/plugins/icase-modulo' ou instaleo plugin diretamente da tela de plugins do WordPress;
2. Ative o plugin na área 'Plugins' do WordPress;
